Adds a widget that displays uploaded media related to a post or page in the sidebar.

== Description ==


Creates a widget that displays attached media that can be added to the sidebar. 

The widget options include:

1. Determine wether links will open in a new window
1. Omit images from being displayed in the output
1. Provide a list of pages to exclude from the output
1. Use a specific widget title (i.e. Related Documents)

== Installation ==

1. Install easily with the WordPress plugin control panel or manually download the plugin and upload the extractedfolder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Widget users can add it to the sidebar by going to the "Widgets" menu under "Appearance" and adding the "Related Documents" widget. Open the widget to configure.
